# 🎮 Free Game Dev — The Big List

> A curated collection of free and open-source tools, assets, and resources for game developers at every level. Quality over quantity — everything here is either **completely free**, **open source**, or has a **free tier worth using**.

**Labels:**
- 🟢 **Free** — free to use, closed source
- 🔵 **Open Source** — source available, may have a paid version
- ⭐ **Beginner Friendly** — great starting point for newcomers
- 🔥 **Community Favorite** — widely used and well-supported

Contributions welcome! See [CONTRIBUTING.md](CONTRIBUTING.md).

| [Open 3D Engine (O3DE)](https://o3de.org/) | 🔵 Apache 2.0 | Windows, Linux | AAA-capable open source engine backed by the Linux Foundation |
